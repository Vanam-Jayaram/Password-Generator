# Password Generator
A simple Python application that generates random passwords based on user-defined lengths.
## Features
* Generate multiple passwords at once
* Custom password lengths
* Includes lowercase letters
* Includes uppercase letters
* Includes numeric characters
* Minimum password length validation
## Technologies Used
* Python 3
* Random Module
## How to Run
1. Open the project folder.
2. Run:
python "Password Generator.py"
3. Enter the number of passwords you want.
4. Enter the desired length for each password.
